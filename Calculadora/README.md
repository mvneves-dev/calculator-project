# Calculadora Simples

Uma calculadora web minimalista e funcional.

## Como usar

1. Abra o arquivo `index.html` no seu navegador
2. Digite o primeiro número
3. Selecione uma operação (+, -, *, /)
4. Digite o segundo número
5. Clique em "Calcular" ou pressione Enter

## Operações disponíveis

- ➕ Adição (+)
- ➖ Subtração (-)
- ✖️ Multiplicação (*)
- ➗ Divisão (/)

## Arquivos

- `index.html` - Interface da calculadora
- `style.css` - Estilos e design
- `script.js` - Lógica das operações
- `calculadora.py` - Versão em Python (terminal)
- `calculadora.js` - Versão em JavaScript (Node.js)

## Requisitos

Nenhum! Basta abrir o HTML no navegador.
